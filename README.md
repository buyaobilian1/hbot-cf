## 项目简介
本项目是基于`cloudflare worker`由`ts`开发的telegram红包雷机器人   
`cloudflare worker`完全免费，所以你并不需要花钱去租用服务器  
本项目遵守 [GPLv3](https://www.gnu.org/licenses/gpl-3.0.html) 开源协议!  

## 功能
- [x] 客服
- [x] 发包
- [x] 抢包
- [x] 结算
- [x] 余额
- [x] 上分
- [ ] 推广
- [ ] 报表
